## Что извлечь

### 1. Текущая база
- Профессия, прошлый и смежный опыт, навыки, технологии
- Доменная экспертиза
- Управленческий опыт
- Образование, текущий доход
- **englishLevel** — нормализуй в CEFR-шкалу:

| Ответ из анкеты | englishLevel |
|-----------------|--------------|
| "Никак, около нуля" / не указан / неясно | "0" |
| Понимает базово, с переводчиком | "A1" |
| "Могу читать/писать, иногда переводчик" | "A2" |
| "Говорю и понимаю не-IT темы" | "B1" |
| "Могу проходить собеседования на английском" | "B2" |
| Свободный, рабочий английский | "C1" |
| Native / билингв | "C2" |

Если неясно — определи по резюме. Если в резюме нет уровня английского и само резюме на русском - ставь A1

### 2. Карьерные цели
- Желаемая зарплата сейчас и через 3-5 лет
- Локация / формат работы
- Готовность к переобучению
- Горизонт цели, приоритет: быстрые деньги / рост / баланс

### 3. Психологический и рабочий профиль
- Тип мышления (определи по ответам: аналитик / креативщик / системщик)
- Интроверт/экстраверт (из ответа про коммуникации)
- Отношение к рутине и неопределенности
- IC vs менеджмент
- Ограничения по энергии / времени / здоровью / семье / географии

### 4. Реальные барьеры

#### 4a. Локация и гражданство (заполни из анкеты)
Из полей `citizenship`, `currentLocation`, `workFormat` извлеки 4 факта:

- **physicalCountry** — страна проживания, нормализованная на английском: "Russia", "Germany", "Argentina", "Kazakhstan"
- **citizenships** — массив гражданств на английском: ["Belarus"], ["Kazakhstan", "Russia"]
- **isRemoteOnly** — true если ищет только удалённую работу, не готов к релокации
- **explicitlyExcludesRU** — true ТОЛЬКО если кандидат явно написал что не хочет работать на РФ-рынок ("только EU", "РФ не рассматриваю"). По умолчанию false.

#### 4b. Остальные ограничения
- **languageBarrier** — достаточен ли английский для целевого рынка (текстом)
- **otherBarriers** — прочие ограничения

### 5. Интерес к направлениям
- Что кандидат указал как интерес
- Его обоснование

### 6. Предыдущие попытки
- Были ли попытки что-то изменить, что пробовал

### 7. Ключевые достижения из резюме
Извлеки конкретные цифры и достижения, которые можно подсветить рекрутерам.
