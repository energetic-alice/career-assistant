# Frontend & Mobile Development - рынок 2025-2026

Источник: StackOverflow Survey, State of JS, LinkedIn Talent Insights

## Фреймворки по спросу
| Фреймворк | Тренд | Конкуренция | Доля вакансий |
|-----------|-------|-------------|---------------|
| React/Next.js | Стабильный | Высокая | ~55% |
| Vue.js | Стабильный | Средняя | ~15% |
| Angular | Стабильный-сужающийся | Средняя | ~20% (enterprise) |
| React Native | Стабильный | Средняя | ~8% |
| Flutter | Растет (+10%) | Низкая-средняя | ~5% |
| Svelte | Растет, нишевый | Низкая | ~2% |

## Ключевые тренды
- **AI-давление на UI-разработку**: AI генерирует компоненты, layouts, CSS -> Junior frontend под угрозой
- **Full-stack тренд**: React/Next.js + Node.js/tRPC -> спрос на full-stack растет
- **Доменная специализация защищает**: iGaming, fintech, e-commerce frontend - премия +15-25%
- **Design Systems**: отдельная специализация, растущий спрос
- **Vue < конкуренция чем React**: при сравнимом спросе, меньше кандидатов на Vue
- **Mobile сужается**: React Native/Flutter заменяют нативную разработку

## Карьерные треки
```
Junior Frontend -> Middle -> Senior Frontend -> Staff Frontend
                                             -> Tech Lead
                                             -> Full-Stack (Node.js)
                           -> React Native -> Mobile Lead
                           -> Design System Lead
```

## AI-риск: СРЕДНИЙ
- Junior/Middle: AI генерирует UI-компоненты, layout, CSS - рутина автоматизируется
- Senior: архитектура, performance, complex UX interactions - AI не заменяет
- Защита: full-stack навыки, доменная экспертиза, Design Systems
- Full-stack (frontend + backend) - самый устойчивый трек

## Рекомендации по переходам
- Frontend -> Full-Stack: естественный переход, +15-20% к зп
- Frontend -> React Native: быстрый переход, мобильные зарплаты выше
- Angular -> React: 2-3 мес переход, расширяет рынок
- Frontend Senior -> Tech Lead / EM: управленческий трек
