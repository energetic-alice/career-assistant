# Remote Work в IT - 2025-2026

Источник: Index.dev, Deel Global Hiring Report, Buffer State of Remote Work, Eurostat

## Статистика remote/hybrid
- **Нидерланды**: 54% работников на удаленке (лидер ЕС)
- **Германия**: 1.7M+ IT-вакансий, ~40% remote/hybrid
- **Финляндия, Швеция**: 45-50% remote
- **Южная Европа**: 25-35% remote (Испания, Италия, Португалия)
- **Глобально**: ~30% tech-работников полностью remote, ~45% hybrid

## Тренды 2025-2026
- **Hybrid = новая норма**: 60% компаний hybrid, 25% full remote, 15% office-only
- **Return-to-office замедлился**: после волны 2023-2024, стабилизация
- **Remote-first компании растут**: GitLab, Automattic, Zapier, Buffer, Deel, Remote.com
- **Geographic arbitrage сужается**: компании внедряют location-based pay
- **Async-first culture**: timezone-compatible > co-located

## Доступность по рынкам для кандидатов из СНГ
| Рынок | Доступность | Условия |
|-------|-------------|---------|
| РФ remote | Высокая | Основной рынок, рублевая зп |
| EU remote (из ЕС) | Высокая | ВНЖ/гражданство + English B1+ |
| EU remote (из РФ) | Низкая | Санкции, банковские ограничения |
| EU remote (из СНГ) | Средняя | EOR/B2B контракт, English B1+ |
| US remote | Очень низкая (из EU) | Timezone conflict, visa |
| US remote (из Americas) | Средняя | B2B контракт, timezone ok |
| APAC remote | Низкая | Нишевый рынок |

## Платформы для remote-поиска
- **LinkedIn**: основной канал для EU/US remote
- **Remote.co, We Work Remotely, RemoteOK**: агрегаторы
- **Toptal, Turing, Arc**: платформы для remote-контрактов
- **Relocate.me**: relocation + remote в EU
- **hh.ru**: РФ/СНГ, фильтр "удаленная работа"

## LinkedIn SSI и remote job search
SSI (Social Selling Index) критичен для EU remote:
- SSI < 20: LinkedIn фактически не работает
- SSI 20-40: видимость ограничена
- SSI 40-60: нормальный уровень
- SSI 60+: активный канал офферов

Для EU рынка SSI < 30 = красный флаг, нужна оптимизация профиля.
Для РФ рынка LinkedIn менее критичен (hh.ru основной).

## Налоговые и юридические аспекты
- **Двойное налогообложение**: проверять соглашения между странами
- **EOR**: Deel, Remote.com берут 500-700 EUR/мес за обслуживание
- **B2B**: в Польше, Чехии - стандартная модель, экономия на налогах 15-25%
- **Permanent establishment risk**: если работодатель в одной стране, а ты в другой
