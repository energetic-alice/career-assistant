## Ключевые тренды
- **DE > DS по найму**: Data Engineering обгоняет Data Science по числу вакансий (+20% YoY)
- **Modern Data Stack premium**: dbt + Snowflake/BigQuery + Airflow = +20-40% к зарплате
- **AI-риск для Junior DA**: рутинные отчеты и дашборды автоматизируются LLM
- **Product Analytics растет**: A/B testing, retention, activation - бизнес-критичные метрики
- **Конвергенция ролей**: Analytics Engineer = DA + DE, самая быстрорастущая роль

## Карьерные треки
```
DA (Junior) -> DA (Senior) -> Lead DA -> Head of Analytics
                            -> Analytics Engineer -> Senior AE
                            -> Product Analyst -> Growth Analyst
                            
DA -> DE -> Senior DE -> Staff DE -> Data Platform Lead
                      -> ML Engineer -> MLOps

Non-IT -> DA/BA (вход) -> через 1-2 года рост в DE, Product Analyst
```

## Порог входа
- **DA (самый низкий)**: SQL + Excel/Sheets + BI-инструмент (Tableau/Power BI/Looker). Вход 3-6 мес.
- **DE**: SQL + Python + Airflow/dbt + облако. Вход 6-12 мес из DA.
- **DS/ML**: статистика + Python + ML frameworks. Вход 12+ мес, нужен матбэкграунд.

## AI-защита
- Junior DA рутина автоматизируется -> защита через доменную экспертизу
- DE pipeline-ы усложняются от AI -> больше инфраструктуры
- Product Analytics = бизнес-контекст, AI не заменяет интерпретацию
